# AWS CloudWatch Alarm Triage

Uses the AWS SDK (boto3 CloudWatch client) to list ALARM-state metrics, fetch metric data points via get_metric_data, and correlate with CloudTrail events. Provides structured incident triage for EC2, RDS, Lambda, and ECS service alarms.
