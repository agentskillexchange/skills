# Monitor Claude Code sessions and tool activity with Claude Code Agent Monitor

Install Claude Code hooks that stream session, tool, subagent, and workflow events into a local dashboard for live observability and review.

## Prerequisites

Claude Code, Node.js 20+, npm, local dashboard server; optional MCP host integration

## Installation

Prerequisites: Node.js 20 or newer and Python 3.6 or newer.

Install and set up from the upstream repository:

- git clone https://github.com/hoangsonww/Claude-Code-Agent-Monitor.git
- cd Claude-Code-Agent-Monitor

Then run the upstream setup task npm run setup, followed by npm run install-hooks to install the Claude Code hooks.

Start the monitor using the upstream documented app command when you are ready to run the dashboard.
