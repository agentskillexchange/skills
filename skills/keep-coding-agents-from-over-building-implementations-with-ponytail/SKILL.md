# Keep coding agents from over-building implementations with Ponytail

Ponytail gives coding agents a portable minimal-implementation ruleset, plugin hooks, and review commands so they reuse existing code, native platform features, and standard library behavior before writing new abstractions.

## Prerequisites

A supported coding-agent host such as Claude Code, Codex, Gemini CLI/Antigravity CLI, OpenCode, OpenClaw, Copilot CLI, Devin CLI, Hermes Agent, Qoder, Swival, Cursor, Windsurf, Cline, or another host that can load the provided rules, plugin, or skill files; Node.js is required for the Claude Code and Codex lifecycle hooks.
