# Shiori Self-Hosted Bookmark Manager with CLI and Web Interface

Shiori is a simple, portable bookmark manager written in Go inspired by Pocket. It provides both a command-line interface and a clean web UI for adding, editing, searching, and organizing bookmarks with automatic offline archive generation and readable content extraction.

## Installation

Install this skill using one of the following methods:

### OpenClaw CLI
```bash
openclaw skills install shiori-self-hosted-bookmark-manager-cli-web
```

### Claude Desktop (MCP)
Add to your Claude Desktop MCP configuration to use this skill with Claude.

### Cursor / Windsurf
Install via the Cursor or Windsurf skill marketplace.

### Manual Installation
Download from the [Agent Skill Exchange](https://agentskillexchange.com/skills/shiori-self-hosted-bookmark-manager-cli-web/) and follow the setup instructions.
