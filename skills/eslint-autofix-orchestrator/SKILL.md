# ESLint Auto-Fix Orchestrator

Runs ESLint with auto-fix capabilities using the ESLint Node.js API and flat config system. Supports custom rule sets, TypeScript via typescript-eslint, and Prettier integration.

## Installation

Use the upstream install or setup path that matches your environment:
- npm init @eslint/config@latest
- npx eslint yourfile.js

Requirements and caveats from upstream:
- To use ESLint, you must have [Node.js](https://nodejs.org/) (^20.19.0, ^22.13.0, or >=24) installed and built with SSL support. (If you are using an official Node.js distribution, SSL is always built in.)
- node-linker=hoisted
