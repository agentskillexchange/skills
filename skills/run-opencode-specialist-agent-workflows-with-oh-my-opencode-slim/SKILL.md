# Run OpenCode specialist-agent workflows with Oh My Opencode Slim

Install an OpenCode orchestration plugin that routes codebase work across specialist agents, background tasks, model presets, bundled skills, and live terminal multiplexers.

## Prerequisites

OpenCode, Bun or Node.js/npx, optional tmux, zellij, herdr, cmux, or kitty for live multiplexer viewing

## Installation

Use the upstream install or setup path that matches your environment:
- npx oh-my-opencode-slim@latest install
- git clone https://github.com/alvinunreal/oh-my-opencode-slim.git ~/repos/oh-my-opencode-slim

Requirements and caveats from upstream:
- The published CLI is a Node-compatible bundle, so npx works too if you don't

Basic usage or getting-started notes:
- **[Council](docs/council.md)** - run multiple models in parallel on the same
- Copy and paste this prompt to your LLM agent (Claude Code, AmpCode, Cursor, etc.):
- bunx oh-my-opencode-slim@latest install
