# Super Productivity Open Source Task Manager with Time Tracking

Super Productivity is a feature-rich, open-source task management application built with TypeScript and Angular. It combines a todo list with time tracking, timeboxing, and a Pomodoro timer into a single unified interface available on Windows, macOS, Linux, Android, and iOS.
Core Capabilities
The application supports planning and categorizing tasks using sub-tasks, projects, and tags with color coding. Its time tracking system generates time sheets and work summaries that can be exported to company time tracking systems. A break reminder prompts users to step away at intervals, and an anti-procrastination feature helps regain perspective during difficult tasks.
Integrations
Super Productivity integrates directly with Jira, Trello, GitHub, GitLab, Gitea, OpenProject, Linear, ClickUp, and Azure DevOps. It can auto-import tasks assigned to you, plan details locally, automatically create work logs, and notify you immediately when something changes upstream. Basic CalDAV integration is also available for calendar sync.
Data and Privacy
The application does not collect any user data and requires no user accounts or registration. Data synchronization across devices is supported via Dropbox and WebDAV. Users can attach notes, files, and project-level bookmarks for links, files, and even commands to tasks and projects.
Agent Integration
AI agents can leverage Super Productivity through its local file-based storage, allowing programmatic task creation, time log export, and project organization. The application supports keyboard-driven workflows with Vim-style navigation and a short syntax system for rapid task entry (e.g., adding tags with #, time estimates with m/h, due dates with @, and project assignment with +).
Installation
Super Productivity is available through multiple channels: Snap, Flatpak, Chocolatey, Scoop, AUR, and direct downloads from GitHub releases. A web version is available at app.super-productivity.com for quick access without installation.
