# 绝境反弹心理术

> 不是鸡汤是方法。屡遭背叛仍信人，尊严第一。

## 一、核心认知

绝境反弹不是天赋，是一种可训练的心理肌肉。你被击倒的次数越多，反弹力越强——前提是你每次都站起来。

## 二、反弹四阶段

### 2.1 震荡期（被击倒）
- 正常反应：愤怒、自我怀疑、想放弃
- 错误做法：压抑情绪、假装没事
- 正确做法：给自己3天崩溃期，然后开始行动

### 2.2 止血期（稳住）
- 先确保生存底线：钱、住、吃
- 不做大决定，只做稳住的事
- 把能控制的控制住，不能控制的先放

### 2.3 反击期（行动）
- 从最小的事开始：洗个澡、打个电话、见个人
- 行动产生信心，信心产生更多行动
- 不要等状态好了再动，动了状态才会好

### 2.4 重建期（升级）
- 复盘：这次亏在哪，下次怎么避
- 不是变冷漠，是变聪明
- 信任可以有，但要有验证

## 三、背叛修复

### 3.1 被背叛后的心理重建
- 不自责：别人背叛你是他的选择，不是你的错
- 不报复：报复消耗的是你自己的时间和精力
- 不遗忘：记住教训，但不要让它变成心结

### 3.2 还能不能信人？
能。但要加验证：
- 信任给第一层，验证给第二层
- 小事先信，大事先验
- 被骗一次是他的问题，被骗两次是你的问题

### 3.3 底线法则
- 底线不是用来妥协的，是用来守护的
- 走之前把该拿的拿回来——这不是小气，是尊严
- 每次妥协底线，下次底线会更低

## 四、绝境决策

| 情况 | 错误决策 | 正确决策 |
|------|---------|---------|
| 钱快没了 | 借高利贷 | 降消费+找收入 |
| 被合伙人坑 | 忍了 | 算清账再走 |
| 工作丢了 | 随便找一个 | 先想清楚要什么 |
| 关系崩了 | 求回来 | 评估值不值得 |

## 五、人生规律

干最多→被辜负→走前拿回底线。

这不是被害妄想，这是对现实的清醒认知。你不亏待别人，但也不让别人亏待你。尊严是底线，不是天花板。
