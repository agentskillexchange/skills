# Orchestrate coding agents across desktop, mobile, and headless sessions with Paseo

Paseo runs a local daemon for Claude Code, Codex, Copilot, OpenCode, and Pi agents so operators can start, monitor, attach to, and hand off coding-agent work from desktop, mobile, web, or CLI clients.

## Prerequisites

At least one configured supported agent CLI such as Claude Code, Codex, GitHub Copilot CLI, OpenCode, or Pi; Paseo desktop app, mobile app, web client, Docker daemon, or the @getpaseo/cli package; local network or pairing access for cross-device clients.
