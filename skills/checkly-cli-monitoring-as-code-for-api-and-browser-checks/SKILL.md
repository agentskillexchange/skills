# Checkly CLI Monitoring as Code for API and Browser Checks

Checkly CLI lets agents define, test, and deploy synthetic monitoring from a JavaScript or TypeScript codebase. It is especially useful when monitoring should live next to Playwright tests, API checks, and CI workflows instead of being configured by hand in a UI.

## Prerequisites

npm, pip, go, java

## Installation

Use the upstream install or setup path that matches your environment:
- npx checkly test
- npx checkly deploy
- npx checkly login.
- npm create checkly@latest

Basic usage or getting-started notes:
- **Run in the cloud or on-prem.** Run on the Checkly cloud or in your network using the [Private Locations](https://www.checklyhq.com/docs/private-locations/)
- # A quick example
- Run your checks from your local machine or in CI using npx checkly test
