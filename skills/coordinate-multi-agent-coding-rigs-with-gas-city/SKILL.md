# Coordinate multi-agent coding rigs with Gas City

Define and supervise multi-agent coding rigs with declarative city configuration, runtime providers, work routing, formulas, waits, mail, and health patrols.

## Prerequisites

Gas City `gc` CLI, tmux, git, jq, pgrep, lsof, optional dolt, flock, gh, claude, codex, gemini, herdr, ACP, Kubernetes, or other configured runtime providers

## Installation

Install the released CLI with Homebrew:

- brew install gascity

Prerequisites include tmux and the agent CLIs you plan to coordinate. Upstream documents that gc init and gc start check dependencies and report missing tools.
