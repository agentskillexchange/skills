# Inspect live web apps and apply code changes with Stagewise

Use Stagewise when a coding agent needs browser context, console/debugger signals, UI inspection, and code-change handoff in one loop.

## Prerequisites

Stagewise app, a running web application, optional connected IDE/codebase, and model provider credentials or a Stagewise account

## Installation

Basic usage or getting-started notes:
- Download stagewise from [stagewise.io](https://stagewise.io) and follow the short onboarding guide to set up your account.
- ## Use your coding subscription
- Bring Your Own Key for all popular model providers — you can also register completely custom providers (including local inference!) and define custom models.
