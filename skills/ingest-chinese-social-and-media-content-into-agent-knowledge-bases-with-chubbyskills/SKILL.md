# Ingest Chinese social and media content into agent knowledge bases with Chubbyskills

Use Chubbyskills when an agent needs installable skills for collecting Chinese social, video, podcast, and article content into Markdown and Obsidian-style knowledge bases.

## Prerequisites

Supported agent runtime, selected Chubbyskills skill folder, optional ffmpeg/yt-dlp/SenseVoice/faster-whisper/DeepSeek API depending on the workflow, local Markdown or Obsidian-style knowledge base

## Installation

Use the upstream install or setup path that matches your environment:
- pip install -r douyin-transcribe/requirements.txt
- brew install ffmpeg yt-dlp # macOS
- pip install -r podcast-transcribe/requirements.txt
- brew install ffmpeg # macOS
