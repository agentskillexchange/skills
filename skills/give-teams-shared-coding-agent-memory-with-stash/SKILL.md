# Give teams shared coding-agent memory with Stash

Connect coding agents to a shared Stash knowledge base so sessions, files, pages, and team context can be searched, cited, updated, and reused across agent runs.

## Prerequisites

uv, Stash CLI package `stashai`, a Stash account or self-hosted Stash instance, repository access, and one or more supported coding agents.

## Installation

Install or set up from the source-backed instructions:

Install with `uv tool install stashai`, then run `stash signin` in the repository you want to connect. For self-hosted use, deploy the documented Docker Compose stack, install the CLI, run `stash signin`, choose Self-host, and enter the self-hosted Stash URL.
