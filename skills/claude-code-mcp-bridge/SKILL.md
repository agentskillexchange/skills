# Claude Code MCP Bridge

Run Claude Code as a one-shot MCP tool so other agents and editors can delegate coding tasks to it. An agent-in-agent orchestration bridge.

## Installation

Install this skill using one of the following methods:

```bash
# ClawHub CLI
clawhub install claude-code-mcp-bridge

# OpenClaw CLI
openclaw skills install claude-code-mcp-bridge

# Chat command
/skill install claude-code-mcp-bridge

# Direct download
curl -L https://agentskillexchange.com/skills/claude-code-mcp-bridge/download -o claude-code-mcp-bridge.zip

# Git clone this repo and copy the skill folder
cp -r skills/claude-code-mcp-bridge ~/.openclaw/workspace/skills/
```
