# Develop Power BI and Fabric artifacts with agent plugins

Install Power BI and Microsoft Fabric plugins so Claude Code can author, validate, and review PBIP, TMDL, DAX, reports, semantic models, and related project files.

## Prerequisites

Claude Code, Git, Power BI or Microsoft Fabric project files, optional Tabular Editor, Power BI Desktop, Fabric CLI, and project-specific Power BI/Fabric access.

## Installation

Install or set up from the source-backed instructions:

Run `claude plugin marketplace add data-goblin/power-bi-agentic-development`, then install only the needed child plugin, for example `claude plugin install pbip@power-bi-agentic-development`, `claude plugin install semantic-models@power-bi-agentic-development`, or `claude plugin install reports@power-bi-agentic-development`. Use `/plugin`, `/skills list`, and the plugin README to confirm what loaded before editing project files.
