# ESLint Rule Auditor

Scans ESLint configuration files and runs eslint –print-config to audit rule conflicts, deprecated rules, and missing plugin dependencies. Integrates with eslint-plugin-import and @typescript-eslint/parser to surface type-aware linting gaps across monorepo workspaces.
