# Oxlint High-Performance JavaScript Linter

Run Oxlint from the Oxc toolchain to lint JavaScript and TypeScript codebases at extreme speed. Written in Rust, Oxlint is 50-100x faster than ESLint and used by Shopify, ByteDance, and Preact.

## Installation

Requirements and caveats from upstream:
- [Rolldown] and [Nuxt] use Oxc for parsing. [Rolldown] also uses Oxc for transformation and minification. [Nova], [swc-node], and [knip] use [oxc_resolver][docs-resolver-url] for module resolution. [Preact], [Shopify],...
- [swc-node]: https://github.com/swc-project/swc-node

Basic usage or getting-started notes:
- **Lint**: [Oxlint](https://oxc.rs/docs/guide/usage/linter) — npx oxlint@latest
- **Format**: [Oxfmt](https://oxc.rs/docs/guide/usage/formatter) — npx oxfmt@latest
- Parse JavaScript and TypeScript: [Parser](https://oxc.rs/docs/guide/usage/parser)
