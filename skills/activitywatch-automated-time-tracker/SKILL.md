# ActivityWatch Cross-Platform Automated Time Tracker and Productivity Analyzer

ActivityWatch is a privacy-first, open-source automated time tracker that records application usage, browser activity, and AFK status across Windows, macOS, and Linux. It provides detailed productivity analytics without sending data to external servers.

## Installation

Requirements and caveats from upstream:
- aw-server (Python) - The current default implementation
- **Client libraries:** Language-specific libraries like aw-client (Python), aw-client-js, and aw-client-rust that wrap REST endpoints for programmatic access
