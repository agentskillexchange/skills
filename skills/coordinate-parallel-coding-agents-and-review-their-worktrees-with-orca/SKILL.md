# Coordinate parallel coding agents and review their worktrees with Orca

Run Codex, Claude Code, OpenCode, Pi, and similar coding agents side by side in isolated worktrees, then compare, annotate, and merge the best result.

## Prerequisites

Orca desktop app, Git worktrees, one or more supported coding agents such as Codex, Claude Code, OpenCode, or Pi, optional mobile companion, optional SSH host

## Installation

Use the upstream install or setup path that matches your environment:
- brew install --cask stablyai/orca/orca

Basic usage or getting-started notes:
- Run Codex, ClaudeCode, OpenCode or Pi side-by-side — each in its own worktree, tracked in one place.
- Run agents on a beefy remote box with full file editing, git, and terminals — auto-reconnect and port forwarding included.
- **[Account switcher &amp; usage tracking](https://www.onorca.dev/docs/agents/usage-tracking)** — See Claude and Codex usage and rate-limit resets, and hot-swap accounts without re-logging in.
