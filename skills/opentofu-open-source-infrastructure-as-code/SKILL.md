# OpenTofu Open Source Infrastructure as Code Tool

OpenTofu is an open-source infrastructure as code tool that lets you declaratively manage cloud and on-premises resources. A CNCF project and community-driven fork of Terraform, it provides execution plans, resource graphs, and change automation for safe infrastructure provisioning.

## Installation

### Install via ClawHub (recommended)
```bash
clawhub install opentofu-open-source-infrastructure-as-code
```

### Install via Git
```bash
git clone https://github.com/agentskillexchange/skills.git
cp -r skills/skills/opentofu-open-source-infrastructure-as-code ~/.openclaw/workspace/skills/
```

### Install via curl
```bash
mkdir -p ~/.openclaw/workspace/skills/opentofu-open-source-infrastructure-as-code
curl -sL https://raw.githubusercontent.com/agentskillexchange/skills/main/skills/opentofu-open-source-infrastructure-as-code/SKILL.md \
  -o ~/.openclaw/workspace/skills/opentofu-open-source-infrastructure-as-code/SKILL.md
```

### Install via OpenClaw CLI
```bash
openclaw skill install opentofu-open-source-infrastructure-as-code
```

### Manual installation
Download the SKILL.md file and place it in `~/.openclaw/workspace/skills/opentofu-open-source-infrastructure-as-code/`
