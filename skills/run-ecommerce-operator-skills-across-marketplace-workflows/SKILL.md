# Run ecommerce operator skills across marketplace workflows

Use eCommerce-Skills to give compatible agents plain-text workflows for product research, listing optimization, review analysis, pricing, marketing, supply-chain checks, and growth planning for online sellers.

## Prerequisites

Node.js and npx skills CLI; compatible Skills host such as OpenClaw, Claude Code, Cursor, Windsurf, Codex, or another markdown-skill-capable agent

## Installation

Install or set up from the source-backed instructions:

Install the full skill pack with npx skills add nexscope-ai/eCommerce-Skills -g, or install one workflow with npx skills add nexscope-ai/eCommerce-Skills --skill -g, such as --skill ecommerce-growth-strategy. Restart or reload the target assistant before invoking the installed ecommerce workflow.
