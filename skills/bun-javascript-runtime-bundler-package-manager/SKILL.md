# Bun All-in-One JavaScript Runtime Bundler and Package Manager

Bun is an incredibly fast all-in-one JavaScript and TypeScript toolkit that serves as a runtime, bundler, test runner, and package manager. Built with Zig and powered by JavaScriptCore, it offers dramatically lower startup times and memory usage compared to Node.js while maintaining broad compatibility.
