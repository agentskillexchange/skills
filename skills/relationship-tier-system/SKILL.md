# 人际关系分级管理系统

> 不是教你怎么社交，是教你怎么守好每一层关系边界。独创五级体系。

## 一、核心认知

大多数人的社交问题不是不会交朋友，是**不会分层**。把所有人当朋友=没有朋友，把所有人当敌人=没有活路。分级管理才是正解。

## 二、五级分类体系

### 🔒🔒🔒🔒🔒 绝交级
- 标准：背叛过底线、伤害过核心利益、不可原谅
- 策略：零接触，信息完全隔离
- 示例：散打老师（5个）、初恋/小徒弟/瑞祥居大股东（4个）
- 铁律：进来的出不去，这是黑名单不是灰名单

### 🔒🔒🔒🔒 警惕级
- 标准：有过失信记录但未触及底线，有限度合作
- 策略：只谈利益不谈感情，每句话留证据
- 示例：大姐（3个）、华海老板（2个）
- 规则：可以共事但不可以共财

### 🔒🔒🔒 有限信任级
- 标准：关系尚可但未经大事考验
- 策略：日常交往正常，核心信息选择性分享
- 规则：观察1年以上且无负面事件可升级

### 🔒🔒 观察级
- 标准：新认识的人，还在了解阶段
- 策略：热情但不深入，多听少说
- 规则：6个月内无异常可升级为有限信任级

### 🔓 开放级
- 标准：普通社交关系，无利益交集
- 策略：正常交往，不涉及金钱和核心隐私
- 规则：一旦涉及利益，自动降入观察级重新评估

## 三、升级与降级规则

### 3.1 升级条件
| 从→到 | 条件 | 观察期 |
|-------|------|--------|
| 观察→有限信任 | 6个月无异常+1次合作顺利 | 6个月 |
| 有限信任→警惕 | 1年以上+3次以上大事考验 | 1年 |
| 警惕→开放 | 近乎不可能，失信记录永久留存 | — |

### 3.2 降级触发
- 涉及金钱纠纷→直接降2级
- 在背后说你坏话→降1级
- 违背承诺→降1级
- 背叛核心利益→直接🔒×5

## 四、信息隔离策略

| 级别 | 可以分享 | 绝对不能分享 |
|------|---------|-------------|
| 🔒×5 | 无 | 一切 |
| 🔒×4 | 公开信息 | 财务/家庭/核心人脉 |
| 🔒×3 | 日常动态 | 收入/底线/决策逻辑 |
| 🔒×2 | 兴趣爱好 | 人脉/资源/计划 |
| 🔓 | 表面社交内容 | 任何实质性信息 |

## 五、踩坑记录

1. 别把所有人当🔓→受伤的都是你
2. 别把警惕级当朋友→该算的账必须算清
3. 升级要慢，降级要快→信任是积累的，崩塌是一瞬间
4. 金钱是最大的试金石→涉及钱，关系立刻现原形

## 六、本质

社交不是交朋友，是**风险管理**。你不需要所有人都喜欢你，你需要知道谁能在关键时刻靠得住。
