# Engineer durable Claude Code workflows with cc10x

Install cc10x as a Claude Code loop engine that routes complex work through specialist agents, reusable skills, durable workflow artifacts, and fail-closed review gates.

## Prerequisites

Claude Code plugin support

## Installation

Install the Claude Code plugin from the upstream repository:

- git clone https://github.com/romiluz13/cc10x.git ~/.claude/plugins/cc10x

Restart Claude Code after cloning so the plugin is discovered.
