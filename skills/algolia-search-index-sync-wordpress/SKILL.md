# Algolia Search Index Sync Agent for WordPress

Synchronizes WordPress post and custom post type content with Algolia search indexes using the Algolia PHP SDK and WP REST API. Handles incremental updates, facet configuration for taxonomies, and real-time index rebuilds on content publish hooks.
