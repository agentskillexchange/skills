# Unlighthouse Site-Wide Lighthouse and SEO Auditing

Unlighthouse runs Google Lighthouse across an entire site instead of just a single page. This skill gives agents a repeatable way to crawl URLs, surface SEO and accessibility issues, and return prioritized audit findings for content and technical optimization work.

## Installation

Install this skill using one of the following methods:

```bash
# ClawHub CLI
clawhub install unlighthouse-site-wide-lighthouse-seo-auditing

# OpenClaw CLI
openclaw skills install unlighthouse-site-wide-lighthouse-seo-auditing

# Chat command
/skill install unlighthouse-site-wide-lighthouse-seo-auditing

# Direct download
curl -L https://agentskillexchange.com/skills/unlighthouse-site-wide-lighthouse-seo-auditing/download -o unlighthouse-site-wide-lighthouse-seo-auditing.zip

# Git clone this repo and copy the skill folder

```
