# Google Cloud Platform CLI gcloud Command-Line Tools

The Google Cloud CLI (gcloud) is the primary command-line tool for interacting with Google Cloud services. It enables developers and DevOps engineers to create and manage GCP resources including Compute Engine, GKE, Cloud Run, Cloud Storage, BigQuery, and IAM from the terminal.
