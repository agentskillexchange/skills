# TypeDoc TypeScript API Documentation Generator

TypeDoc is the standard documentation generator for TypeScript projects. It reads TypeScript source code and JSDoc comments to produce structured HTML documentation or JSON models. It powers API reference generation for thousands of TypeScript libraries.
