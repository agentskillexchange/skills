# Expose CLI agent accounts through an API-compatible local proxy with CLIProxyAPI

Run CLIProxyAPI when an operator needs a local OpenAI, Gemini, Claude, Codex, or Grok-compatible API facade over authenticated CLI agent access for repeatable agent and tool integrations.

## Prerequisites

CLIProxyAPI binary or source build; supported local CLI/OAuth credentials for the providers being proxied; compatible OpenAI/Gemini/Claude/Codex client
