# Run phased coding-agent delivery workflows with AgentSys

Use AgentSys to install and run structured coding-agent pipelines for task discovery, delivery preparation, drift checks, review, release, and shipping.

## Prerequisites

Node/npm, git, and a supported coding-agent runtime such as Claude Code, Codex CLI, OpenCode, Cursor, or Kiro

## Installation

Install or set up from the source-backed instructions:

Install from npm with npm install -g agentsys, then use the AgentSys installer or marketplace commands documented upstream to add the desired workflows for Claude Code, Codex CLI, OpenCode, Cursor, or Kiro.
