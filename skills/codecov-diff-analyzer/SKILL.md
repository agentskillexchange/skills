# Codecov Diff Analyzer

Uses the Codecov API v2 (/api/v2/github/{owner}/repos/{repo}/commits) to analyze coverage diffs on pull requests. Surfaces uncovered lines in changed files and compares against Codecov YAML threshold configurations.
