# Install Chinese Claude Code skills for engineering workflows

Use claude-code-skills-zh to install Chinese-language Claude Code skills for repeatable code review, README writing, API testing, refactoring, dependency auditing, and performance profiling workflows.

## Prerequisites

Claude Code with skills support; local shell access to clone or download the repository; selected skill folders copied into ~/.claude/skills/

## Installation

Install or set up from the source-backed instructions:

Run git clone https://github.com/laolaoshiren/claude-code-skills-zh.git and copy selected folders from claude-code-skills-zh/skills/ into ~/.claude/skills/, or copy all upstream original skill folders with cp -r claude-code-skills-zh/skills/* ~/.claude/skills/.
