# Serve production agentic RAG through R2R

Ingest documents, expose hybrid retrieval and agentic RAG APIs, and give agents a production retrieval service with citations and reviewable document operations.

## Prerequisites

Python or Docker, R2R server, API keys for the selected model provider, optional JavaScript SDK

## Installation

Choose whichever fits your setup:

1. Copy this skill folder into your local skills directory.
2. Clone the repo and symlink or copy the skill into your agent workspace.
3. Add the repo as a git submodule if you manage shared skills centrally.
4. Install it through your internal provisioning or packaging workflow.
5. Download the folder directly from GitHub and place it in your skills collection.

Install command or upstream instructions:

```
Install with `pip install r2r`, set the required model provider API key, run `python -m r2r.serve`, then connect agents or clients to the R2R API for document ingestion, search, RAG, and agentic retrieval calls.
```
