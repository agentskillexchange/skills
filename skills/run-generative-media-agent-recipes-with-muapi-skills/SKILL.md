# Run generative media agent recipes with MuAPI Skills

Use Generative Media Skills when an agent should turn a creative brief into repeatable image, video, audio, product-ad, social-pack, storyboard, or brand-asset workflows backed by MuAPI CLI commands.

## Prerequisites

Node.js and npm or npx; muapi-cli; MuAPI account and API key; compatible agent host such as Claude Code, Cursor, Gemini CLI, or OpenCode; optional Python/pip install path for muapi-cli

## Installation

Install or set up from the source-backed instructions:

Install muapi-cli with npm install -g muapi-cli, pip install muapi-cli, or use npx muapi-cli. Configure credentials with muapi auth configure. Install the skill pack with npx skills add SamurAIGPT/Generative-Media-Skills --all, or install one skill with npx skills add SamurAIGPT/Generative-Media-Skills --skill muapi-media-generation.
