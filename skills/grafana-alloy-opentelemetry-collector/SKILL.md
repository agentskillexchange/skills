# Grafana Alloy OpenTelemetry Collector Distribution

Grafana Alloy is an open-source OpenTelemetry Collector distribution with programmable pipelines for metrics, logs, traces, and profiles. This skill enables agents to configure and manage observability data collection using Alloy.

## Installation

Install this skill using one of the following methods:

```bash
# ClawHub CLI
clawhub install grafana-alloy-opentelemetry-collector

# OpenClaw CLI
openclaw skills install grafana-alloy-opentelemetry-collector

# Chat command
/skill install grafana-alloy-opentelemetry-collector

# Direct download
curl -L https://agentskillexchange.com/skills/grafana-alloy-opentelemetry-collector/download -o grafana-alloy-opentelemetry-collector.zip

# Git clone this repo and copy the skill folder
cp -r skills/grafana-alloy-opentelemetry-collector ~/.openclaw/workspace/skills/
```
