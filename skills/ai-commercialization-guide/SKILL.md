# AI智能体商业化实战指南

> 所有内容均来自亲身实践，非理论推演。做不到的不写，没跑通的不说。

## 一、商业化核心公式

```
OpenClaw + 一切行业 = 印钞机
```

底层逻辑：AI智能体是基础设施，行业知识是变现杠杆。你不需要懂所有行业，你只需要帮行业的人把AI用起来。

## 二、变现漏斗设计

```
免费Skill引流 → 付费远程部署 → 月费托管/行业包
    ↓                  ↓                ↓
  建信任            核心利润           持续收入
```

### 2.1 免费Skill引流
- 在扣子技能商店/虾评发布免费Skill
- Skill内容=你最强的实战经验
- 目标：让潜在客户看到你的能力，主动找你

| 套餐 | 价格 | 包含内容 |
|------|------|---------|
| 基础版 | ¥599 | OC部署+1个渠道接入+基础配置 |

| 企业版 | ¥2,999起 | 全渠道+定制Skill+专属部署+优先支持 |

**增值服务：**
- 额外渠道接入：¥200/个

## 三、Skill上架实战

### 3.1 扣子技能商店
1. 访问 https://www.coze.cn/skills
2. 开通商户：选择"小微商户"，门槛最低
3. 商户信息填写：
   - 商户名称：个人姓名
   - 商户简称：XX AI工作室
   - 行业：互联网服务→计算机软件
   - 行业资质：可不上传（小微商户）
4. 创建Skill：在"我的技能"页面创建
5. ⚠️ 技能商店现为申请制（2026.5.29起），需填写资质问卷
   - 专业领域选"其他"，补充说明"AI智能体部署与多渠道网关配置"
   - 需要专业资质/机构背书/行业影响力才能通过
   - 个人开发者建议先上虾评，攒够案例再申请

### 3.2 虾评平台（推荐先上）
1. 注册：`POST https://xiaping.coze.com/api/auth/register`
2. 上传Skill ZIP（含SKILL.md）
3. 默认进入30天众测期（trial）
4. 转正条件：5条≥4星评测 或 2位A4-1+等级用户好评
5. 转正后正式上架，开始产生收益

### 3.3 Skill包装三原则
1. **只写亲测可行的** — 做不到的别写，这是铁律
2. **标题用动作短语** — "部署OpenClaw到飞书"比"OC介绍"好
3. **description写清场景** — 用户3秒内判断要不要下载

## 九、踩坑记录

### 9.1 不要写做不到的
曾经把Telegram写进渠道列表，实际国内用不了。客户问起来答不上，搬石头砸自己脚。铁律：**只写亲手跑通过的**。

### 9.2 渠道信息不要写太细
免费/门槛/价格不要放在公开文档里。写太细=帮竞争对手做调研。主动权在自己手里，私聊再谈。

### 9.3 扣子技能商店资质
个人开发者目前很难通过资质审核（需专业资质/机构背书/行业影响力）。别死磕，先上虾评攒案例。

### 9.4 虾评评测规则
- 评测必须20字以上
- 评测要通过API提交（网页上没有评论入口）
- 完整评测（含维度评分）奖励更多虾米
- LLM会自动评评测质量，别写水评
