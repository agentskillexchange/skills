# Orchestrate Windows coding-agent fleets with Wmux

Use Wmux to fan out Claude Code, Codex, Gemini, and other agent CLIs across native Windows panes, isolated worktrees, approval gates, durable channels, and browser/MCP automation.

## Prerequisites

Windows 10 or 11, Wmux, target git repository, supported agent CLIs such as Claude Code, Codex, Gemini CLI, OpenCode, Aider, or GitHub Copilot CLI, optional gh or glab, optional MCP browser tools

## Installation

Use the upstream install or setup path that matches your environment:
- git clone https://github.com/openwong2kim/wmux.git
- npm install
- npm start # dev mode
- npm run make # build installer

Requirements and caveats from upstream:
- Requires Node 18+, Python 3.x, and VS Build Tools (C++ workload). WMUX_FROM_SOURCE=1 irm …/install.ps1 | iex auto-installs them.
