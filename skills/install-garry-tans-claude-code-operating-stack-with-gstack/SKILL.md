# Install Garry Tan's Claude Code operating stack with gstack

Use gstack to install Garry Tan's opinionated Claude Code operating stack, including configuration, MCP servers, and supporting toolchain in a single setup step.

Operator workflow: clone the gstack repository, run the install script, review the applied Claude Code configuration and MCP server entries, and verify the stack is active inside a Claude Code session.

## Prerequisites

- Claude Code (`@anthropic-ai/claude-code`)
- Node.js / npm

## Setup

Follow the installation steps in the gstack repository:

```
git clone https://github.com/garrytan/gstack
cd gstack
./install.sh
```
