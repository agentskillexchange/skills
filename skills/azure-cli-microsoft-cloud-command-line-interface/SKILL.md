# Azure CLI Microsoft Cloud Command-Line Interface

The Azure CLI is a set of commands used to create and manage Azure resources from the command line. It provides a consistent interface for interacting with Azure services across Windows, macOS, and Linux, enabling infrastructure automation, DevOps pipelines, and resource management.
