# Cursor AI-Powered Code Editor

Cursor is an AI-first code editor built on VS Code that deeply integrates large language models into the coding workflow. It enables developers to write, edit, and understand code through natural language interactions, with features like AI-powered autocomplete, codebase-aware chat, and agent mode.
