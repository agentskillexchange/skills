# Generate reusable site adapters from live browser behavior with OpenCLI

OpenCLI gives an agent a concrete workflow for turning a website, browser session, desktop app, or local tool into a reusable command surface. The sharpest skill-shaped use is adapter creation: explore a live site, capture the behavior that matters, and generate a stable CLI command for repeatable future use. Invoke this when a site or tool does not already have a dependable API or existing command, and you want to convert a one-off browser task into a deterministic operator command. That is different from using the product normally, because the goal is not to browse manually, it is to crystallize the workflow into a reusable agent-facing interface. The boundary is adapter generation and browser-backed command synthesis. It is not a generic browser tool listing, not just a product homepage, and not a catch-all automation framework card.
