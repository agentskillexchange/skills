# Stancl Tenancy v3 — Multi-Tenant Laravel

Builds multi-tenant Laravel SaaS applications with stancl/tenancy v3. Best when you need domain/subdomain/path-based tenant identification, multi-database or single-database isolation, tenant-aware queues/cache/filesystems/Redis, production bootstrappers, configuration per-tenant, and seamless integration with Horizon, Scout, Livewire, and Spatie packages. Covers the full package API surface including event system, JobPipeline, console commands, testing patterns, and critical production gotchas.
