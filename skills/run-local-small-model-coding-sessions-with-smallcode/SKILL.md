# Run local small-model coding sessions with SmallCode

Use SmallCode as a terminal-native coding agent for 8B-35B local models, with budget-managed context, forgiving tool parsing, TODO-driven planning, and patch-based edits.

## Prerequisites

Node.js/npm, a local 8B-35B coding model or compatible local model runtime, terminal access to the target project
