# AutoGen Microsoft Multi-Agent Conversation Framework

AutoGen is an open-source framework by Microsoft for building multi-agent AI systems where multiple AI agents collaborate, converse, and execute tasks. It enables complex workflows by orchestrating LLM-powered agents with tool-use, code execution, and human-in-the-loop capabilities.
