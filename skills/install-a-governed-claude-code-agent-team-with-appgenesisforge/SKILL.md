# Install a governed Claude Code agent team with AppGenesisForge

Add a Claude Code team scaffold with defined roles, stage gates, mandatory skills, blocking hooks, and Definition of Done checklists for app delivery.

## Prerequisites

Claude Code v2.1.154 or newer, git, macOS or Linux, optional iTerm2/tmux for split-view teammate mode

## Installation

Use the upstream install or setup path that matches your environment:
- git clone https://github.com/pcliangx/AppGenesisForge.git

Requirements and caveats from upstream:
- **卡分屏**：模板默认 teammateMode: "auto"——iTerm2 / tmux 会话自动分屏、其余终端静默 in-process，不影响功能，无需配置。想强制 iTerm2 分屏 fail-loud 才在用户级 ~/.claude/settings.json 设 "iterm2"（需 pip install it2 + iTerm2 Python API，缺 it2 会显式报错）。
