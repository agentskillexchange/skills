# GitHub Actions Workflow Debugger

Analyzes GitHub Actions workflow YAML files and queries the GitHub REST API (/repos/{owner}/{repo}/actions/runs) to diagnose failed runs. Inspects step logs, runner environment variables, and action version pinning for security and reliability.
