# Automate Lark and Feishu docs, sheets, mail, calendar, meetings, and chat tasks from agents with Lark CLI

Use Lark CLI when an agent needs to act on concrete Lark or Feishu work objects such as messages, docs, sheets, mail, meetings, approvals, or calendar events, instead of using Lark manually or hand-rolling API calls.
