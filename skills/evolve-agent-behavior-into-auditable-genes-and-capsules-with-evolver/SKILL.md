# Evolve agent behavior into auditable Genes and Capsules with Evolver

Turn recurring agent lessons, logs, and strategy tweaks into reviewable GEP evolution artifacts that can be applied across supported coding-agent runtimes.

## Prerequisites

Node.js 18+, Git, npm, a git-initialized project, optional hooks for Cursor, Claude Code, Codex, Kiro, opencode, or OpenClaw

## Installation

Choose whichever fits your setup:

1. Copy this skill folder into your local skills directory.
2. Clone the repo and symlink or copy the skill into your agent workspace.
3. Add the repo as a git submodule if you manage shared skills centrally.
4. Install it through your internal provisioning or packaging workflow.
5. Download the folder directly from GitHub and place it in your skills collection.

Install command or upstream instructions:

```
npm install -g @evomap/evolver, then run evolver inside a git-initialized project. Use evolver --review for human approval or evolver setup-hooks --platform= to integrate with a supported agent runtime.
```
