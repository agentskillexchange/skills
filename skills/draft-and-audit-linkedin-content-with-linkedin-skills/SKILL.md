# Draft and audit LinkedIn content with LinkedIn Skills

Use LinkedIn Skills when an agent needs to draft posts, comments, replies, audits, profile rewrites, or content plans while preserving explicit approval before publishing.

## Prerequisites

Claude Code, Codex, OpenClaw, Hermes, or another SKILL.md-compatible agent; optional Apify token for read-side LinkedIn data; optional Publora API credentials for scheduled publishing

## Installation

Install or set up from the source-backed instructions:

For Codex CLI, run codex plugin marketplace add sergebulaev/linkedin-skills and codex plugin add linkedin-skills@linkedin-skills. For Claude Code, run /plugin marketplace add sergebulaev/linkedin-skills and /plugin install linkedin-skills@linkedin-skills. For generic SKILL.md agents, run npx skills add sergebulaev/linkedin-skills or clone the repository and point the agent at the skills directory.
