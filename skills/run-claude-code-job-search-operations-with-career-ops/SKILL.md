# Run Claude Code job-search operations with Career-Ops

Use Career-Ops to run a repeatable Claude Code powered job-search workflow: evaluate listings, tailor resumes, generate applications, and review outputs before sending anything.

## Prerequisites

Claude Code or compatible CLI agent, Node.js/npm, Career-Ops CLI/dashboard, Playwright where browser automation is used, candidate profile and resume source materials

## Installation

Choose whichever fits your setup:

1. Copy this skill folder into your local skills directory.
2. Clone the repo and symlink or copy the skill into your agent workspace.
3. Add the repo as a git submodule if you manage shared skills centrally.
4. Install it through your internal provisioning or packaging workflow.
5. Download the folder directly from GitHub and place it in your skills collection.

Install command or upstream instructions:

```
Install Career-Ops from the upstream npm package or GitHub release, configure the candidate profile and documents, run the Claude Code skill modes for job discovery, evaluation, resume tailoring, and PDF generation, then review every generated application artifact before external submission.
```
