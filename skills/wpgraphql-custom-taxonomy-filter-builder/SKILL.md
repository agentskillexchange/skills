# WPGraphQL Custom Taxonomy Filter Builder

Generates custom WPGraphQL where-clause filters for any registered WordPress taxonomy using register_graphql_field and WP_Tax_Query. Supports nested AND/OR taxonomy queries with term slug and ID resolution.
