# Prepare meetings and follow-ups from local work memory with Rowboat

Use Rowboat to build local work memory from email, calendar, notes, and web context, then generate meeting briefs, follow-ups, docs, and decks.

## Prerequisites

Rowboat desktop app, connected email/calendar or notes sources, local Markdown vault, optional MCP or Composio tools, and optional Deepgram, ElevenLabs, or Exa API keys
