# Apply an anti-AI-slop design workflow to pages, components, audits, and redesigns with Hallmark

Give coding agents a design skill for greenfield UI, component work, audits, redesigns, and visual DNA extraction from screenshots or URLs.

## Prerequisites

A supported coding agent such as Claude Code, Cursor, or Codex; project files or a target screenshot/URL for audit, study, redesign, or implementation work

## Installation

Use the upstream install or setup path that matches your environment:
- npx skills add nutlope/hallmark

Basic usage or getting-started notes:
- Re-run any time to update. Or copy [SKILL.md](skills/hallmark/SKILL.md) + [references/](skills/hallmark/references/) into:
- **Claude Code**: ~/.claude/skills/hallmark/
- **Cursor**: .cursor/rules/hallmark.mdc (body of SKILL.md, no frontmatter)
