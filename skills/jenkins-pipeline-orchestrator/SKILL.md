# Jenkins Pipeline Orchestrator

Controls Jenkins CI via the Jenkins REST API and jenkinsfile-runner. Triggers builds through POST /job/{name}/build, retrieves console output from /job/{name}/{build}/consoleText, and manages pipeline configurations using the Pipeline Model Definition Plugin and Jenkins Configuration as Code (JCasC).
