# Build and refactor Copilot Studio agents as YAML from coding-agent workflows

Use Skills for Copilot Studio when you want a coding agent to create, edit, validate, test, or troubleshoot Copilot Studio agent definitions as files. The plugin exposes focused commands for authoring YAML, syncing agent content, running tests, and debugging routing or validation issues from Claude Code, GitHub Copilot CLI, or VS Code. Invoke this instead of using Copilot Studio normally when the advantage is code-first agent authoring and repeatable validation in your editor or terminal. The scope is bounded to Copilot Studio YAML workflows and related test or troubleshooting commands, not a generic Copilot Studio listing.
