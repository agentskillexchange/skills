# Run Parallel Divergent Ideation With ADHD

Use ADHD to fan out isolated reasoning branches for open-ended coding-agent decisions, then score, prune, and deepen the strongest options.

## Prerequisites

Node.js 18+, npx skills CLI or npm/npx, target agent runtime such as Claude Code, Codex, Cursor, Gemini CLI, OpenCode, Windsurf, Cline, Aider, or compatible Agent Skills host

## Installation

Prerequisite: Node.js 18 or newer.

Install with the upstream skills CLI:

- npx skills add UditAkhourii/adhd

For Codex, upstream also documents forcing the Codex target:

- npx skills add UditAkhourii/adhd -a codex -g
