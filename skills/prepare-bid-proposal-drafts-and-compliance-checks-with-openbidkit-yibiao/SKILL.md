# Prepare bid proposal drafts and compliance checks with OpenBidKit Yibiao

Use OpenBidKit Yibiao to turn tender materials, company knowledge, and model-provider settings into reviewable bid proposal drafts with reuse and risk-check workspaces.

## Prerequisites

OpenBidKit Yibiao desktop app or local development build, OpenAI-compatible API or local model provider such as Ollama or LM Studio, tender documents, reusable company proposal materials

5. Download the folder directly from GitHub and place it in your skills collection.

Install command or upstream instructions:

```
Download the latest OpenBidKit Yibiao release from https://github.com/FB208/OpenBidKit_Yibiao/releases, install or run the desktop build, configure an OpenAI-compatible API or local model provider, then create a local bid workspace with tender files and company knowledge-base materials before generating and reviewing proposal sections.
```
