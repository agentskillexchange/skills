# Monitor live coding-agent sessions with Abtop

Run a local read-only terminal dashboard for Claude Code, Codex CLI, and OpenCode sessions to watch tokens, context, rate limits, child processes, and orphan ports.

## Prerequisites

abtop CLI, Claude Code, Codex CLI, or OpenCode sessions; sqlite3 for OpenCode monitoring on Linux; optional tmux, cmux, or iTerm2 terminal jump support
