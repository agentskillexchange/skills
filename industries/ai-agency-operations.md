# 🧭 AI Agency Operations & FDE Workflows

Client-facing AI delivery, forward deployed engineering workflows, browser automation, implementation systems, documentation, spreadsheets, proposals, and client handoff workflows.

- Live page: https://agentskillexchange.com/industry-skills/#ai-agency-operations
- Homepage access: Curated Collections on https://agentskillexchange.com/

## Who this is for

- AI agency founders, forward deployed engineers, automation consultants, client delivery leads, and solutions engineers packaging repeatable AI-enabled services for clients.
- Teams delivering supervised client implementations that need browser automation, agent app scaffolding, design-to-code context, implementation artifacts, and release checks.

## Jobs covered

- Audit client sites, docs, workflows, and product surfaces for implementation planning.
- Build repeatable client automation workflows across browsers, agent apps, tools, and internal systems.
- Package findings into proposals, reports, docs, spreadsheets, and handoff artifacts.
- Implement browser automation, design-to-code, workflow delivery, and client-facing agent stacks.
- Monitor delivery quality and reusable agency workflows.

## Workflow Stacks

- **Client audit sprint:** Capture site context → summarize findings → build report → deliver action plan
- **Automation implementation:** Map workflow → connect tools → trigger handoffs → validate outputs
- **Client handoff packet:** Draft proposal → repair workbook → write docs → record workflow → handoff implementation
